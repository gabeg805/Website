Projects

Here you'll find the most recent projects I've been working. If you want to see more, check out my other projects on Github.

Elysia

Login Manager

Elysia is a highly configurable login manager, written primarily in C, that utilizes the GTK 3.0 library to draw text boxes, buttons, etc. that will add to the login experience.

Atlas

Status Bar

Atlas is an alternative status bar to the Dynamic Window Manager's (DWM). It is written in C and utilizes the GTK 3.0 library to display icons for battery level, brightness, wifi strength, and more.

Aria

Notification Bubble

Aria is a highly configurable notification bubble, as an alternative to notification packages such as "notify-osd". It is written entirely in C and utilizes the GTK 3.0 library in order to draw the bubble(s) on the screen.
